   - Новый модуль с полным кодом (400+ строк)

3. **Интеграция в PDE и Optimization** (~2 часа)

4. **Реалистичные граничные условия** (~1.5 часа)
   - Непрерывность на границах раздела
   - Диrichlet на датчиках
   - REST референция

5. **Интеграция с MNE-Python** (~2 часа)
   - `BEMHeadModel` класс на Python
   - Загрузка реальных BEM моделей
